# Remote Repository

## Clone a repo

```bash
$ git clone <url>

$ git clone <url> <name> # provide the name you want the folder to be called
```

## Push changes to origin

```bash
$ git push origin
```

## Pull the latest changes

```bash
$ git pull origin
```
